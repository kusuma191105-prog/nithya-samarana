
# 🕉️ Nitya Smarana Dashboard

## 🚀 Live Demo
👉 **Try the app here:** https://5ddfb278-6e2b-459a-b42c-5adbd19b299c-00-2kxmrc360vnd6.sisko.replit.dev

Track your daily devotional japa practice with analytics and streaks.

### Features
- 📿 Log daily mantra count
- 🔥 Auto-calculated streak tracking
- 📊 Pie chart for mantra-wise distribution
- 📈 Bar chart for daily progress
- 💾 Data saved in JSON format

### Tech Stack
Python, Streamlit, Pandas, Plotly

### How to Run
1. Install requirements: `pip install -r requirements.txt`
2. Run app: `streamlit run app.py`
